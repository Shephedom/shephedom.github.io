编译UE源码

发表于 2024-12-13

一、将github账户与Epic Game账户绑定

  1. 打开Epic Game个人页面–应用与账户–连接github
  2. 在关联账户中点击Github
  3. 然后邮箱会收到Epic Games发来的邀请，点击后面的View Invitation后，在接下来的页面点击Join Epic Games

二、克隆代码

右键预存放源码的文件夹，点击Git Bash Here, 输入
git clone https://github.com/EpicGames/UnrealEngine
放一些我下载时出现过的问题，以及解决方法：

  1. github账户和EpicGame账户未关联上

fatal: repository 'https://github.com/EpicGames/UnrealEngine/' not found

  1. 网络连接问题或代理设置不正确

fatal: unable to access 'https://github.com/EpicGames/UnrealEngine.git/': OpenSSL SSL_read: SSL_ERROR_SYSCALL
参考文章 https://blog.csdn.net/qq_43546721/article/details/139506583
打开系统的代理设置，在代理服务器中将端口设置为7890，设置 Git 使用本地代理：
git config --global http.proxy http://127.0.0.1:7890

  1. 项目过大

方法一：修改缓冲区大小 和 下载速度限制

方法二：只克隆仓库最新一次的commit（最保险的办法）

git clone https://github.com/EpicGames/UnrealEngine --depth 1

下载完成后，输入cd UnrealEngine

定位版本 git checkout 5.5.0-release
（我只克隆仓库最新一次的commit，在github上显示最新的版本号是5.5.0）

三、下载依赖文件，生成工程文件

在本地仓库文件夹的顶部路径中输入cmd，回车，打开命令提示符，输入 Setup.bat

下载完成后，同样的方法，运行 GenerateProjectFile.bats 生成工程文件

四、编译运行

打开解决方案文件(.sln)，将 Engine 文件夹中的 UE5 设为启动项目，调试代码

五、新建UE Project并打包

编译启动引擎后，新建一个项目，点击 平台-Windows-打包项目
